### 01 - Cybersecurity-Bedrohungen: Arten, Techniken

- **Arten von Bedrohungen**: Viren, Trojaner, Ransomware, Phishing, DDoS, XSS, SQL-Injektion
- **Techniken**: Malware, Social Engineering, Man-in-the-Middle, Denial of Service (DoS), Backdoors
- **Schutzmaßnahmen**: Firewalls, Verschlüsselung, Sicherheitsupdates, Multi-Faktor-Authentifizierung (MFA), Antivirus-Software
- **Ziel**: Datenklau, Systemkompromittierung, Unterbrechung des Dienstes

---

### 02 - Social Engineering

- **Definition**: Manipulation von Menschen, um Zugang zu Systemen/Daten zu erhalten
- **Techniken**: Phishing, Spear-Phishing, Pretexting, Baiting, Quizzes
- **Ziel**: Vertrauen ausnutzen, um sensible Informationen zu stehlen
- **Schutzmaßnahmen**: Mitarbeiterschulungen, Sicherheitsrichtlinien, Awareness-Programme

---

### 03 - Schädliche Software & Cyberangriffe

- **Schädliche Software**: Viren, Würmer, Trojaner, Spyware, Ransomware
- **Angriffstechniken**: SQL-Injektionen, Cross-Site Scripting (XSS), Drive-by-Downloads
- **Ziel**: Datenklau, Systemkompromittierung, Erpressung (Ransomware)
- **Schutzmaßnahmen**: Echtzeit-Scanning, regelmäßige Updates, Backup-Strategien, Sandbox-Umgebung

---

### 04 - Drahtlose & Mobile Sicherheitsbedrohungen

- **Bedrohungen**: Man-in-the-Middle (MITM), Bluejacking, Bluesnarfing, SMiShing, WEP/WPA Schwächen
- **Techniken**: Angriffe auf Wi-Fi, Bluetooth, Mobilfunknetzwerke
- **Schutzmaßnahmen**: Verwendung von VPNs, WPA2, AES-Verschlüsselung, Gerätesperre, Zwei-Faktor-Authentifizierung (2FA)

---

### 05 - Webanwendungs-Sicherheitsbedrohungen

- **Bedrohungen**: Cross-Site Scripting (XSS), SQL-Injektion, CSRF, Directory Traversal, Remote Code Execution (RCE)
- **Angriffstechniken**: Manipulation von Benutzereingaben, unsichere APIs, Pufferüberläufe
- **Schutzmaßnahmen**: Eingabevalidierung, sichere Codierung, regelmäßige Patches, HTTPS, Content Security Policy (CSP)
